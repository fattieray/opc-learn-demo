## ⏳ 待完成

### 后端云函数部署

由于 CloudBase CLI 3.1.0 版本的命令有所变化，云函数需要通过控制台部署。

#### 方法 1: 通过控制台部署（推荐）

1. 访问: https://console.cloud.tencent.com/tcb/scf/index
2. 选择环境: opc-learn-backend-5eut7hd7ad9071
3. 点击"新建云函数"
4. 配置:
   - **函数名称**: opc-api
   - **运行环境**: Python 3.8
   - **内存**: 512MB
   - **超时时间**: 30秒
   - **部署方式**: 本地上传 zip
5. 打包后端代码:
   ```bash
   cd backend
   zip -r ../opc-api.zip . -x "*.pyc" "__pycache__/*"
   ```
6. 上传 opc-api.zip
7. 配置环境变量:
   - DATABASE_URL: sqlite+aiosqlite:///tmp/opc_learn.db
   - SECRET_KEY: your-secret-key-change-this
   - CORS_ORIGINS: ["https://opc-learn-backend-5eut7hd7ad9071-1258123235.tcloudbaseapp.com"]

#### 方法 2: 使用旧版 CLI

```bash
# 安装旧版 CLI
npm install -g @cloudbase/cli@1.9.0

# 部署云函数
tcb function:create opc-api --runtime Python3.8 --memory 512 --timeout 30
tcb function:deploy opc-api --dir backend
```
