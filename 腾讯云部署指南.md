## 🚀 开始部署

### 步骤 1: 安装 CloudBase CLI

```bash
# 全局安装
npm install -g @cloudbase/cli

# 验证安装
tcb --version
```

### 步骤 2: 登录腾讯云

```bash
# 扫码登录（会弹出二维码）
tcb login

# 按照提示用微信扫描二维码
```

### 步骤 3: 创建 CloudBase 环境

```bash
# 方式 1: 通过控制台创建（推荐）
# 访问: https://console.cloud.tencent.com/tcb/env

# 方式 2: 使用 CLI 创建
tcb env:create opc-learn-prod

# 记录你的环境 ID（类似: opc-learn-xxx）
```

### 步骤 4: 初始化项目

```bash
cd opc-learn-demo

# 初始化 CloudBase 项目
tcb init

# 选择:
# - 关联环境: 选择刚创建的环境
# - 应用名称: opc-learn
# - 部署目录: .
```

### 步骤 5: 配置前端部署

创建 `cloudbaserc.json`：

```json
{
  "envId": "你的环境ID",
  "region": "ap-shanghai",
  "functionRoot": "./backend",
  "functions": [
    {
      "name": "opc-api",
      "timeout": 30,
      "runtime": "Python3.8",
      "memorySize": 512,
      "handler": "main.handler"
    }
  ],
  "framework": {
    "name": "opc-learn",
    "plugins": {
      "frontend": {
        "use": "@cloudbase/framework-plugin-website",
        "inputs": {
          "buildCommand": "npm run build",
          "outputPath": ".next",
          "cloudPath": "/",
          "headers": [
            {
              "source": "/_next/static/**",
              "headers": [
                {
                  "key": "Cache-Control",
                  "value": "public, max-age=31536000, immutable"
                }
              ]
            }
          ]
        }
      },
      "function": {
        "use": "@cloudbase/framework-plugin-function",
        "inputs": {
          "functionRootPath": "./backend"
        }
      }
    }
  }
}
```

### 步骤 6: 改造后端为云函数

创建 `backend/main.py`（云函数入口）：

```python
from mangum import Mangum
from app.main import app

# 将 FastAPI 应用包装为云函数
handler = Mangum(app, lifespan="off")
```

更新 `backend/requirements.txt`：

```
mangum==0.17.0
fastapi==0.104.1
uvicorn==0.24.0
pydantic==2.5.0
pydantic-settings==2.1.0
aiosqlite==0.19.0
PyJWT==2.8.0
passlib==1.7.4
bcrypt==4.0.1
python-multipart==0.0.6
```

### 步骤 7: 配置环境变量

```bash
# 设置前端环境变量
cat > .env.production << EOF
NEXT_PUBLIC_API_URL=https://opc-api-xxx.service.tcloudbase.net
EOF

# 后端环境变量在云函数控制台配置
```

### 步骤 8: 部署

```bash
# 一键部署（前端 + 后端）
tcb deploy

# 或分别部署
tcb hosting:deploy .next  # 部署前端
tcb function:deploy opc-api  # 部署后端
```

### 步骤 9: 配置数据库

```bash
# 创建云数据库集合
tcb database:create courses
tcb database:create users
tcb database:create missions
tcb database:create circles

# 导入初始数据
tcb database:import courses ./data/courses.json
```

### 步骤 10: 配置自定义域名（可选）

```bash
# 在 CloudBase 控制台
# 1. 环境 -> 域名管理
# 2. 添加域名: opc-learn.cn
# 3. 配置 DNS 解析
# 4. 等待 SSL 证书签发（约 10 分钟）
```

## 📝 详细配置说明

### 后端云函数改造

#### 1. 创建云函数入口文件

```python
# backend/index.py
from mangum import Mangum
from app.main import app

# Mangum 将 ASGI 应用转换为云函数格式
handler = Mangum(app, lifespan="off")
```

#### 2. 修改数据库配置

云函数环境需要使用云数据库，而不是本地 SQLite：

```python
# backend/app/core/config.py
from pydantic_settings import BaseSettings
from typing import List
import os

class Settings(BaseSettings):
    APP_NAME: str = "OPC Learn Demo"
    APP_VERSION: str = "0.1.0"
    ENVIRONMENT: str = "production"
    
    # 数据库 - 使用腾讯云云数据库
    DATABASE_URL: str = os.getenv(
        "DATABASE_URL",
        "mysql+pymysql://user:password@host:3306/opc_learn"
    )
    
    # JWT
    SECRET_KEY: str = os.getenv("SECRET_KEY", "dev-secret-key-change-in-production")
    ACCESS_TOKEN_EXPIRE_MINUTES: int = 60 * 24 * 7
    
    # CORS
    CORS_ORIGINS: List[str] = [
        "https://opc-learn-xxx.service.tcloudbase.net",
        "https://opc-learn.cn",
    ]
    
    model_config = {"env_file": ".env"}

settings = Settings()
```

#### 3. 更新依赖

```bash
cd backend

# 添加云数据库驱动
echo "pymysql==1.1.0" >> requirements.txt
echo "cryptography==41.0.0" >> requirements.txt

# 添加 Mangum（FastAPI 云函数适配器）
echo "mangum==0.17.0" >> requirements.txt
```

## 🎯 完整自动化脚本

### 一键部署脚本

```bash
#!/bin/bash
# deploy-tencent.sh

set -e

echo "🚀 开始部署 OPC Learning Hub 到腾讯云..."

# 1. 检查依赖
echo "📦 检查依赖..."
if ! command -v tcb &> /dev/null; then
    echo "安装 CloudBase CLI..."
    npm install -g @cloudbase/cli
fi

# 2. 登录
echo "🔐 登录腾讯云..."
tcb login

# 3. 设置环境变量
echo "⚙️ 配置环境变量..."
read -p "请输入 CloudBase 环境 ID: " ENV_ID
echo "NEXT_PUBLIC_API_URL=https://opc-api-$ENV_ID.service.tcloudbase.net" > .env.production

# 4. 构建前端
echo "🔨 构建前端..."
npm install
npm run build

# 5. 配置后端
echo "📝 配置后端..."
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cd ..

# 6. 创建云函数
echo "☁️ 创建云函数..."
tcb function:create --name opc-api --runtime Python3.8 --memory 512 --timeout 30

# 7. 部署前端
echo "🌐 部署前端..."
tcb hosting:deploy .next

# 8. 部署后端
echo "🔧 部署后端..."
tcb function:deploy opc-api

# 9. 配置数据库
echo "🗄️ 配置数据库..."
tcb database:create courses
tcb database:create users
tcb database:create missions
tcb database:create circles

# 10. 完成
echo ""
echo "✅ 部署完成！"
echo "🌐 前端地址: https://$ENV_ID.service.tcloudbase.net"
echo "🔧 后端地址: https://opc-api-$ENV_ID.service.tcloudbase.net"
echo ""
echo "📝 下一步："
echo "  1. 访问前端地址测试"
echo "  2. 配置自定义域名（可选）"
echo "  3. 导入初始数据"
```

使用方式：

```bash
chmod +x deploy-tencent.sh
./deploy-tencent.sh
```

## 🔧 常见问题

### Q1: 云函数冷启动慢？

**A:** 云函数有 1-3 秒冷启动，可以：
1. 设置定时预热（每 5 分钟调用一次）
2. 增加内存（512MB → 1024MB）
3. 使用云函数并发功能

### Q2: 数据库连接失败？

**A:** 检查：
1. 数据库白名单是否添加云函数
2. 数据库连接字符串是否正确
3. 数据库用户名密码是否正确

### Q3: CORS 错误？

**A:** 更新 CORS_ORIGINS：
```python
CORS_ORIGINS: List[str] = [
    "https://你的环境ID.service.tcloudbase.net",
    "https://你的域名.com",
]
```

### Q4: 静态资源 404？

**A:** Next.js 静态资源路径：
```javascript
// next.config.js
const nextConfig = {
  assetPrefix: '/_next',
}
```
