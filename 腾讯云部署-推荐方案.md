# 🚀 腾讯云部署 - 推荐方案

## ⚠️ 当前问题

CloudBase 静态托管**不支持** Next.js 的动态路由功能，因为：
- Next.js 需要 Node.js 服务器运行
- 静态托管只能放纯静态文件（HTML/CSS/JS）
- 你的项目有动态路由 `/circles/[id]` 和 `/courses/[id]`

---

## ✅ 推荐方案：腾讯云轻量应用服务器

### 为什么选择轻量服务器？

1. ✅ **完美支持 Next.js** - 可以运行 Node.js 应用
2. ✅ **价格超低** - 99元/年（2核4G，5M带宽）
3. ✅ **简单易用** - 控制台简洁，有应用镜像
4. ✅ **完全控制** - 和 VPS 一样，想怎么配置就怎么配置
5. ✅ **一个服务器搞定前后端** - 前端 3000 端口 + 后端 9000 端口

## 📋 部署步骤（20 分钟）

### 步骤 1: 购买轻量服务器（5 分钟）

1. 访问: https://cloud.tencent.com/act/pro/lighthouse
2. 选择配置:
   - **地域**: 上海（推荐）
   - **套餐**: 2核4G 5M带宽
   - **系统**: Ubuntu 22.04
   - **价格**: 99元/年（新用户）
3. 完成购买

### 步骤 2: 获取服务器信息（1 分钟）

1. 访问: https://console.cloud.tencent.com/lighthouse/instance/index
2. 找到你的服务器
3. 记录 **公网 IP**
4. 记录 **密码**（或设置密码）

### 步骤 3: 运行一键部署脚本（10 分钟）

```bash
# SSH 连接到服务器
ssh ubuntu@<你的公网IP>

# 运行一键部署脚本
curl -sSL <脚本URL> | bash
```
