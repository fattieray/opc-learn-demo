如果你选择 ECS（我推荐的第一步），这是最快上手的步骤：

### 1. 购买（5分钟）

```
阿里云控制台 -> ECS -> 创建实例
- 地域: 北京
- 规格: 2核4G (ecs.t6-c1m2.large)
- 系统: Ubuntu 22.04
- 磁盘: 40GB
- 带宽: 5Mbps 按量
```

### 2. 一键部署脚本（10分钟）

我已经为你准备好了完整的部署脚本，购买后只需：

```bash
# SSH 连接到服务器
ssh root@<你的ECS公网IP>

# 运行一键部署脚本
curl -sSL https://raw.githubusercontent.com/fattieray/opc-learn-demo/main/deploy-aliyun.sh | bash
```

### 3. 配置域名（5分钟）

```
域名控制台 -> 添加解析 -> 指向 ECS 公网 IP
```
