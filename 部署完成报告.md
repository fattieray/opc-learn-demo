## ⚠️ 重要说明

### Render 免费套餐限制
1. **自动休眠**: 15 分钟无请求后服务会休眠
   - 首次访问需要 10-30 秒唤醒
   - 唤醒后正常使用

2. **数据存储**: SQLite 存储在临时目录
   - 服务重启后数据会丢失
   - 可重新运行 seed API 恢复数据
   - 生产环境建议升级到付费套餐或使用 PostgreSQL

3. **构建限制**: 免费套餐不支持 Rust 编译
   - 已通过 `--prefer-binary` 解决
   - 使用预编译的 Python 包
