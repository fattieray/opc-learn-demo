# 🚀 腾讯云部署 - 快速开始

## ⚡ 5 分钟快速部署

### 前提条件

- ✅ 已安装 Node.js 18+
- ✅ 已安装 npm
- ✅ 有腾讯云账号（可微信扫码注册）

---

## 📋 部署步骤

### 步骤 1: 注册腾讯云（2 分钟）

1. 访问 [腾讯云官网](https://cloud.tencent.com/)
2. 点击"免费注册"
3. 使用微信扫码注册
4. 完成实名认证（需要 1-2 分钟）

### 步骤 2: 创建 CloudBase 环境（2 分钟）

1. 访问 [CloudBase 控制台](https://console.cloud.tencent.com/tcb/env)
2. 点击"新建环境"
3. 填写信息：
   - **环境 ID**: `opc-learn-xxx`（自定义，如 `opc-learn-prod`）
   - **环境名称**: `OPC Learn Production`
   - **地域**: 上海（推荐）
4. 点击"确定"
5. **记录环境 ID**（后面会用到）

### 步骤 3: 运行部署脚本（5 分钟）

```bash
# 进入项目目录
cd opc-learn-demo

# 运行一键部署脚本
./deploy-tencent.sh
```

**脚本会自动完成：**
- ✅ 安装 CloudBase CLI
- ✅ 微信扫码登录
- ✅ 安装项目依赖
- ✅ 构建前端
- ✅ 部署前端到静态托管
- ✅ 创建和部署后端云函数
- ✅ 配置数据库集合

### 步骤 4: 测试访问（1 分钟）

部署完成后，脚本会显示访问地址：

```
🌐 前端: https://你的环境ID.service.tcloudbase.net
🔧 后端: https://你的环境ID.service.tcloudbase.net/api/v1/
```

在浏览器中打开前端地址，测试功能！

## 🎯 手动部署（可选）

如果不想使用脚本，可以手动执行：

### 1. 安装 CloudBase CLI

```bash
npm install -g @cloudbase/cli
```

### 2. 登录

```bash
tcb login
# 用微信扫描二维码
```

### 3. 构建前端

```bash
cd opc-learn-demo
npm install
npm run build
```

### 4. 部署前端

```bash
tcb hosting:deploy .next
```

### 5. 部署后端

```bash
# 创建云函数
tcb function:create --name opc-api --runtime Python3.8 --memory 512 --timeout 30

# 部署云函数
tcb function:deploy opc-api --dir backend
```

## 🔧 常见问题

### Q1: 部署失败，提示权限错误？

**A:** 确保：
1. 已完成腾讯云实名认证
2. 已登录 CloudBase CLI（`tcb login`）
3. 有 CloudBase 环境

### Q2: 前端页面空白？

**A:** 检查：
1. 浏览器控制台是否有错误
2. 环境变量 `NEXT_PUBLIC_API_URL` 是否正确
3. 是否成功构建（`.next` 目录存在）

### Q3: 后端 API 返回 502？

**A:** 等待 1-2 分钟，云函数首次部署需要初始化。

### Q4: CORS 错误？

**A:** 云函数的 CORS 配置在 `backend/app/core/config.py` 中，确保包含你的域名。

### Q5: 数据库连接失败？

**A:** 云函数环境使用 `/tmp/opc_learn.db` 作为 SQLite 数据库路径。

## 📝 自定义域名（可选）

如果需要绑定自己的域名：

### 1. 在 CloudBase 控制台

1. 环境 → 域名管理
2. 添加域名（如 `opc-learn.cn`）
3. 复制 CNAME 记录

### 2. 在域名控制台

1. 添加 DNS 解析
2. 类型: CNAME
3. 主机记录: `@` 或 `www`
4. 记录值: 复制的 CNAME 地址

### 3. 等待生效

DNS 解析通常 10-30 分钟生效，SSL 证书自动签发。
